Deep Learning Technology

End to end deep learning with both image and text data to train a highly accurate phishing detection model. The model has similar visibility (actually greater visibility) to a human and if trained correctly can learn skills of a cybersecurity expert. It is as if you have a cybersecurity expert watching over you at all times.

Features

  • Real time detection of phishing pages.
  • Easy to setup and use (autonomous).
  • Increased detection accuracy compare to other offerings.
  • Countinous improvments (higher quality data and model) adapting to the current ladscape of threats.
  • Potential support for Arabic content (phishing) by including Arabic pages in training.

Get Started

We are still in the prototyping phase (under active development). Please check out the following prototyping repos and reach out through github (by creating an issue).

Pharec Webextension Repo

Experimental code (not stable) of the phishing image classification model used in a browser extension. Currently works on firefox and can be converted to work on other browsers.

Pharec Model Training Repo

Deep image CNN classification model training experiments (notebooks) of the phishing site classifier. Also, starter utils for dataset collection
